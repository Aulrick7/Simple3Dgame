Extra features added:
1. Health packs - Colliding with this health packs will restore all your health.
2. Animation - There is an animation to each individual action including both player and enemies.
3. jumpads/launchpads - An object that leaps the player to great heights.
4. Melee Combat - the player can fight the enemies by either punching or kicking the enemies.
5. quit button - that you can press to quit the game
6. Sounds on certain actions - like the player walking, running etc. and the launchpad. There is also a song
